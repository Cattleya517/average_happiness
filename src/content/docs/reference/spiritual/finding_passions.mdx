# 尋找熱情

### 迷你實驗法 from 做自己的生命設計師

1. 思考三個以上你『完全沒有想過』自己會做的活動（小實驗）
e.g. 淨灘、擔任志工、跳廣場舞、去二手書店待一下午，跟超過三年沒有聯絡的朋友說話，任何事情...

2. 不事先思考任何自己到底可能喜歡或不喜歡這個活動，先做就對了，反正不喜歡當次結束就沒了

3. 活動結束後紀錄自己的感受是什麼
如果沒有感覺，那就放棄，至少你知道你對特定種類的活動沒有興趣。
如果有一點感覺，覺得有點享受或好玩，回想這個小實驗到底是什麼樣的特性引發了你的興趣。
e.g. 跟很多接觸人讓我很開心，我可以歸類跟整理東西讓我很享受，幫助別人讓我感覺很好...

Note.

(1) 數量優先，能想到越多種小實驗越好
(2) Novelty優先，之前從來沒有做過也不曾思考自己可能會去做的活動優先
(3) 成本越低越好，若行有餘力也是可以花錢報名課程，但是最好不要先對這些小實驗有過多的commitment
